# UniTasks 📝

A smart task manager with real-time reminders built using HTML, CSS, and JavaScript.

## Features
- ✅ Add, complete, and delete tasks
- ⏰ Set custom reminder intervals (every N hours)
- 🔔 Browser notifications for incomplete tasks
- 🔊 Sound alerts when reminder fires
- 💾 Tasks saved locally (no data lost on refresh)

## How to Use
1. Enter a task in the input box
2. Set how often you want to be reminded (in hours)
3. Click **Add Task**
4. Allow notifications when the browser asks
5. Get reminded automatically until the task is done!

## Tech Stack
- HTML, CSS, JavaScript
- Web Notifications API
- localStorage
